# External Skills

我们使用但不维护的第三方 skill。不在此 repo 中。

## ClawHub

安装：`openclaw skills install <slug>`
更新：`openclaw skills update --all`

| Skill | Slug | 用途 |
|-------|------|------|
| ffmpeg-cli | ffmpeg-cli | 音视频处理 |
| grok-search | grok-search | Grok 搜索 X/Web |
| mermaid-architect | mermaid-architect | 架构图生成 |
| seo | seo | SEO 优化指南 |
| show-my-ip | show-my-ip | 查公网 IP |
| snap | snap | 网页截图 |
| tailscale | tailscale | Tailscale 组网 |
| ui-ux-pro-max-2 | ui-ux-pro-max-2 | UI/UX 设计指南 |
| vercel | vercel | Vercel 部署 |
