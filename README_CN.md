# OpenClaw Skills — 我们的合集

40 个生产级 AI agent skill，全部原创或有实质性修改。

## 这个 Repo 是什么

只包含**我们自己写的或重写过的** skill。外部/第三方 skill 通过 ClawHub 或手动安装管理 — 见 [EXTERNAL.md](EXTERNAL.md)。

详细文档见 [README.md](README.md)（英文）。

## 快速开始

**已有环境：** 通过 post-commit hook 自动同步到 `~/.openclaw/skills/`。

**新环境：**
```bash
git clone git@github.com:mfang0126/openclaw-skills.git
cd openclaw-skills
bash setup.sh
```
