## [2026-03-21] 初始化 — 历史已知问题汇总

以下是从社区收集的常见错误，作为初始知识库：

1. **tools 写成数组** → 应该是对象 `{ "tools": { ... } }`
2. **tools.allow 覆盖默认工具集** → 用 `alsoAllow` 追加
3. **Agent 直接 Write 修改 openclaw.json** → 必须用 `config.patch`
4. **allowFrom 写成单字符串** → 必须是数组 `["id"]`
5. **agents.list 写成对象** → 必须是数组 `[{ "id": "xxx" }]`
6. **未知字段导致启动失败** → `.strict()` 校验不允许任何多余字段
7. **Docker token 不同步** → .env 和 openclaw.json 里的 token 必须一致
8. **dmPolicy="open" 没加 allowFrom: ["*"]** → 所有 DM 被静默丢弃
9. **heartbeat.every 格式错** → 必须是 "5m"、"1h" 等时长字符串
10. **env 变量名不一致** → 无报错静默失败，工具名必须精确匹配
